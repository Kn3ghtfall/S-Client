# S-Client  
**Blooket Cheats Menu**

S-Client is a single-script cheat client for Blooket, designed to run directly in the browser. It delivers a unified hacking menu built as one integrated codebase.

---

## Overview
- Single-piece codebase (not modular)  
- Explicitly designed for hacking and cheating in Blooket  
- Runs entirely client-side  
- Executable via:
  - UserScript managers (Tampermonkey, Violentmonkey, etc.)
  - Browser developer console  

All functionality is bundled into one script to keep execution simple and consistent.

---

## Capabilities
- Game manipulation and exploit-based features  
- Bypassing normal gameplay restrictions  
- Client-side control over game mechanics  
- Centralized menu for managing hacks  
- Compatible with multiple Blooket game modes  

---

## Execution Methods

### UserScript
Load S-Client into a UserScript manager and run it while on Blooket.

### Browser Console
Paste and execute the script directly in the browser developer console while on Blooket.
